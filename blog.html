Questions about DOM

What is Document Object Model (DOM)?

The Document Object Model is a cross-platform and language-independent interface that treats an HTML or XML document as a tree structure wherein each node is an object representing a part of the document. The DOM represents a document with a logical tree.

How do you select an element from DOM?

There are multiple ways to select elements from the DOM (Document Object Model) in JavaScript, including: getElementById: Selects an element by its ID. This is the simplest and fastest way to select a single element. getElementsByClassName: Selects elements by their class name. getElementsByTagName: Selects elements by their tag name. querySelector: Selects the first element that matches a specified CSS selector. querySelectorAll: Selects all elements that match a specified CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where you attach a single event listener to a parent element, allowing it to handle events triggered on any of its child elements, effectively "delegating" event handling to a higher level in the DOM tree

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM in JavaScript, you first need to select the element using methods like getElementById, getElementsByClassName, or querySelector, then access its attributes using setAttribute and getAttribute for attributes, and use the style property to directly modify its CSS styles inline
